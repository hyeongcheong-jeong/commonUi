auto-caption을 사용할 경우 table에 data-acption-auto="true" 값을 추가해 준다.

표의 제목이 있는경우는 class box_table을 이용하여 제목 + table로 구조를 분리 시켜야 한다

표의 제목이 없는경우는 title 값을 가지고 오며 data-table-title속성에 값을 넣어주면 해당 값을 제목으로 설정한다.
이경우에는 구조를 분리 시키지 않아도 된다.

auto-caption을 사용하면 기존에 입력된 caption 값을 지우고 js에서 해당값을 찾아 변경해준다
